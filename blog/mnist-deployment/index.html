This deployment uses a simple densely connected neural network to make predictions. The backend was implemented using a Lambda function connected to the front-end using the AWS Gateway API service. The model is stored in a private S3 bucket.

Within the Lambda function, a Python script is used to call Tensorflow lite, load the model and make the prediction, returning a list with 10 elements. A layer containing Tensorflow lite was created using Docker and the AWS Linux image. The code within the Lambda function, as well as the front-end can be found in my Github. The front-end is in the 'personal_website' repository, the backend is in the 'mnist' repository.
